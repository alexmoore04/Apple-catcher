# Apple Catcher 🍎

A 2D arcade-style catching game built from scratch in C++. The player controls a basket to catch falling apples across progressively harder levels.

## About

This project was built as part of my first year game programming coursework at the University of Brighton. It demonstrates core game programming fundamentals including game loop architecture, object management, collision detection, and level progression.

## Features

- Player-controlled basket with keyboard input
- Falling objects with increasing speed across levels
- Score tracking and level progression
- Built using a custom C++ game framework

## Tech

- **Language:** C++
- **IDE:** Visual Studio
- **Platform:** Windows

## How to Run

1. Clone the repository
2. Open `CI411_CW2_01.sln` in Visual Studio
3. Build and run in Debug or Release mode
